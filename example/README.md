# Command Palette Kit Example

Full Flutter example for `command_palette_kit`.

The example is generated with Flutter platform scaffolding for Android, iOS,
Linux, macOS, web, and Windows. It demonstrates:

- opening the palette with Cmd+K, Ctrl+K, or a button;
- command search and keyboard navigation;
- quick actions;
- command execution through `CommandActionExecutor`;
- changing app theme from a command.

Run it from this directory:

```sh
flutter run
```

Select a target platform with `flutter devices` and `flutter run -d <device>`.
